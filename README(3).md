# 📚 AI Study Assistant Chatbot

An AI-powered study assistant built with **Python**, **LangChain**, and
the **Groq API**. It explains concepts in simple language, provides
examples, explains code line by line, and saves chat history for future
reference.

## ✨ Features

-   AI-powered study assistant
-   Simple and beginner-friendly explanations
-   Code explanation line by line
-   Conversation memory
-   Chat history logging
-   Built-in commands (`help`, `history`, `clear`, `exit`)
-   Powered by Groq Llama 3.3 70B

## 🛠 Technologies

-   Python
-   LangChain
-   Groq API
-   Llama 3.3 70B Versatile

## 📂 Project Structure

``` text
├── chatbot.py
├── chat_history.txt
└── README.md
```

## 📦 Installation

``` bash
pip install langchain-groq
```

Replace the API key in `chatbot.py` with your own Groq API key.

## ▶️ Run

``` bash
python chatbot.py
```

## 💬 Commands

  Command   Description
  --------- ---------------------------
  help      Show available commands
  history   View chat history
  clear     Reset conversation memory
  exit      Close the chatbot
